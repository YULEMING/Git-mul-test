# 文字总结

## 小型团队

1. fork一份主仓库 --> 自己的github账户
1. clone自己账户的仓库 --> 本地
1. 本地修改后push --> 自己账户的仓库
1. 用VSCode可以用界面推送
1. 接着在github自己账户的仓库里：pull requests -> new pull request -> create pull request. 就可以把刚才的修改提交到主仓库
1. 主仓库管理员在pull requests里可以合并刚才提交的request
1. 开发时需要用到其他人开发的模块可以手动pull，或者点VSCode左下角![img1](./img1.png)
